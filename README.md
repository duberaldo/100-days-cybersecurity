## Day 2: Network Protocols and Essential Ports

Today I learned about logical ports and core network protocols that operate within the TCP/IP layers.

### Key Protocols Studied:
*   **HTTP (Port 80) vs HTTPS (Port 443):** Understood the critical security difference between plaintext traffic and encrypted (SSL/TLS) traffic.
*   **DNS (Port 53):** The internet's directory, translating human-readable domain names into IP addresses.
*   **FTP (Ports 20/21):** Dedicated to file transfer, utilizing port 21 for connection control and port 20 for data.
*   **SSH (Port 22):** Provides secure, encrypted remote terminal access to servers.

### Practical Application:
*   Used the `nslookup` command to analyze how DNS resolution queries work in real-time.
